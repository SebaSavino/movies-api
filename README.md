# Movies API

## Project setup

1. Create an “.env” file based on the “.env.template” file

2. Install dependencies
    ```bash
    # Dependencies
    $ npm install
    ```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
